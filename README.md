# 26-SSP-TradingAgent
26년 여름학기 ai서비스 프로젝트

# Description
트레이딩 전용 터미널과 편의성 기능을 제공하는 LLM 기반 에이전트 프로그램입니다.

투자 시 참고가능한 매크로지표, 시장에 대한 평가, 현재 시장을 이끄는 주도 섹터등의 정보를 제공하는 모니터링 웹 서비스로, **자연어** 를 통해 정보를 탐색하고 요청할 수 있습니다.
이 과정에서 본인에게 필요한 것들을 북마크할 수 있으며 북마크 된 정보는 RAG화 되이 나중에 더 나은 답변을 제공하는데 사용됩니다.
